General Purpose Registers on x86

- rsp = Stack pointer
- rax = Return value
- rdi = 1st argument
- rsi = 2nd argument
- rdx = 3rd argument
- rcx = 4th argument
- r8 = 5th argument
- r9 = 6th argument
- r10, r11 = Callee-owned
- rbx, rbp, r12 - r15 = Caller-owned

Instruction suffixes (AT&T syntax)

- b = byte
- w = word
- l = doubleword
- q = quadword
